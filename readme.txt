INSTALLATION:

* Open the file config.php and edit the details for your mySQL-database.

* Upload all files to your webserver.

* Open install.php in your browser and follow the instructions.

* The installation should now be finished and you can log in to the admin tool with
  the password generated for you.

You can now add records to your collecion
